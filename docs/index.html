Setup & Installation

1. Prerequisites

  • Node.js 18+ and pnpm
  • Discord bot token
  • PostgreSQL (or use bundled Docker Compose)
  • YouTube cookies file (for music)

2. Environment Setup

cp .env.example .env

Fill in required values:

  • DISCORD_TOKEN - Your bot token
  • DISCORD_CLIENT_ID - Your bot client ID
  • DATABASE_URL - PostgreSQL connection string
  • YOUTUBE_COOKIES - Youtube cookies string

3. Using Docker (Recommended)

pnpm docker:up

This will start the bot and PostgreSQL in Docker containers.

Check logs with: pnpm docker:logs

4. Manual Installation

pnpm install
pnpm dev

Run the bot locally without Docker.
