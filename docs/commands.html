Full Command Reference

🎵 Music Commands

/play <URL or query>

Play a song from YouTube, Spotify, or search by title.

/skip

Skip the currently playing track.

/pause

Pause the current playback.

/resume

Resume paused playback.

/stop

Stop playback and clear the queue.

/queue

Display the current music queue.

/now

Show the currently playing track with duration and queue size.

/shuffle

Shuffle the music queue.

/remove <index>

Remove a specific track from the queue by index.

/clear

Clear all tracks from the queue.

🔁 Loop Modes

/loop off

Disable loop mode - play once and stop.

/loop track

Loop the currently playing track.

/loop queue

Loop the entire queue.

/loop autoplay

Enable autoplay - bot adds similar tracks when queue ends.

📝 Lyrics

/lyrics [title] [artist]

Display lyrics for the currently playing track or search for specific lyrics.

🎤 Text-to-Speech

/tts <text>

Convert text to speech and play in the voice channel. Bot must be connected.

🔍 Search

/search <query>

Search for songs and display top 5 results.

⚙️ Admin Commands

/botstats

Display bot statistics and uptime information.

/guildinfo

Show information about the current server (guild).

🛠️ Utility Commands

/help

Display help information and available commands.

/ping

Check bot latency and response time.

/server

Display server status and information.

🔗 Integrations

/pia

Search novel from NovelPia Global
