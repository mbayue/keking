Privacy Policy

This policy explains what Keking processes, what it stores, and how data is used when the bot is added to a Discord server.

Keking is a Discord bot that provides music playback, text-to-speech, utility commands, social-link preview assistance, and server helper features.

1. Information We Process

Keking may process the following information when used inside a Discord server:

  • Server IDs
  • Server names
  • User-submitted slash command inputs
  • Message content in servers where the bot is installed, only when needed for supported bot features
  • Voice channel state needed to join and play audio
  • Interaction metadata required for slash commands and button interactions

2. How Information Is Used

Keking uses this information only to provide bot functionality inside Discord servers, including:

  • Responding to slash commands
  • Playing music and text-to-speech audio in voice channels
  • Detecting supported social-media links in messages and replying with alternative preview links
  • Responding to direct bot mentions where implemented in the bot logic
  • Storing limited guild-level operational data for setup and command deployment

3. Stored Data

Keking stores limited server-level operational data outside Discord only when database storage is enabled. This may include:

  • Guild ID
  • Guild name
  • Command deployment hash
  • Join and update timestamps

Keking does not intentionally store message content, presence history, or member-profile datasets in its database as part of normal operation.

4. Message Content

Keking may read message content in servers only for features that require it, such as supported social-link preview replies and mention-based responses. Message content is processed in real time to perform those features and is not used for advertising or profiling.

5. AI and Model Training

Keking does not use message content or other Discord data to train machine-learning or AI models.

6. Data Sharing

Keking does not sell user data and does not share Discord data with third parties except where necessary to provide requested functionality, such as querying supported external services for music, lyrics, or NovelPia search results.

7. Data Retention

Operational guild data is retained only as long as needed to support the bot's functionality. If the bot leaves a server and database support is enabled, related stored guild records may be deleted.

8. Your Choices

Server owners and administrators control whether to add Keking to their servers. If you no longer want Keking to process data in your server, remove the bot from the server.

9. Contact

For questions about this Privacy Policy or data handling, contact the bot operator through the project repository or support contact associated with the bot listing.
